Trabajos.

.Desafío 1 Mi Curriculum Vitae en HTML {

Esta actividad está diseñada para poner a prueba y consolidar todo lo que has aprendido hasta ahora en el curso. Tu tarea es crear un currículum utilizando HTML. Recuerda aplicar las bases fundamentales de HTML y las etiquetas que hemos explorado en nuestras lecciones. No dudes en incluir cualquier información que consideres importante para tu currículum.

}

.Desafío 2 - Construyendo un landing page {

En este desafío evaluaremos nuestros conocimientos de CSS. Para ello, se implementará CSS utilizando hojas de estilo externas al documento HTML y de base para uniformizar el diseño del sitio. Además, deberás estructurar una página en términos de activos del proyecto, e implementar bordes, márgenes, padding y selectores de etiqueta o clase según sea necesario.

}
Mirar Repositorio
.Desafío 3 - Iguana page {

El objetivo de este desafío es evaluar y validar nuestros conocimientos de posicionamiento en CSS. Para lograrlo, deberás distinguir los valores asociados a display (inline, block, inline-block), utilizar floats para posicionar texto e imágenes, y resolver el problema del colapso de floats.

}
Mirar Repositorio
.Desafío 4 - Cuppon {

El propósito de este desafío es evaluar nuestra habilidad con Bootstrap. Para lograrlo, deberás usar el sistema de rejillas de Bootstrap para organizar el contenido de manera efectiva, creando una página web que se adapte a diferentes dispositivos. Además, tendrás que aplicar las clases de Bootstrap en los componentes de tarjetas y formularios, utilizando la documentación disponible.

}

.Desafío 5 - Github {

El propósito de este reto es evaluar nuestro conocimiento de Github. Para lograrlo, deberás usar las etapas del control de versiones con GIT para gestionar un repositorio de versiones, y el proceso de transferencia del código versionado mediante una conexión SSH, para la administración de un repositorio remoto.
